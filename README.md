## 📝 How to Contribute

We welcome contributions from the community! To add a new hackathon or competition:

1. Fork this repository
2. Add the competition to the appropriate category table
3. Ensure the following details are included:
   - Competition name
   - Prize amount (USD or local currency)
   - Frequency (Annual / Ongoing / etc.)
   - Eligibility (bachelor's students, open to all, etc.)
   - Official website
4. Submit a pull request

**Contribution Guidelines:**
- Only add competitions with verified prizes of at least $500 USD equivalent
- Must be legitimate and recurring (or well-established)
- Must be relevant to bachelor's level students
- Include only official website links
